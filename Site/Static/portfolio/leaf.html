Website remodel

In the spring of 2018, I attended General Assembly's Web Development Immersive program to go accelerate my web development skills and get me headed in the direction I wanted to be going in terms of being a well-rounded Web Developer.

I crafted this website for my favorite restaurant in Boulder Colorado, Leaf Vegetarian Restaurant. My goal for the project was to take the information that their website already had and put it into a more modern and usable framework.

The technologies I used for this project are HTML5 | CSS3 | ReactJS | Express | Node.js | Google FireBase | SendGrid API
